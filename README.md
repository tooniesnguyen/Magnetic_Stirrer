# Magnetic Stirrer
![image](./images/micro.jpg)
## Hardware required
- STM32f103c8t6
- H-bridge l298n
- LM2596
- Adapter 12V
- JGA25-370 DC Geared Motor (Any type)
- Potentiometer
- Grove - 4-Digit Display
